# Pre-requisite
- node js
- python
- virtualenv

# Frontend
- cd frontend
- npm install
- npm run build

# Run frontend application
- go to browser : http://localhost:3000

# Backend
- virtualenv venv
- source venv/bin/activate        (Mac)
- cd backend
- pip install Django
- python manage.py runserver

# Run backend application
- go to browser : http://localhost:8000
